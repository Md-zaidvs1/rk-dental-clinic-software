# RK DENTAL CLINIC SUITE v1.0 RC1 - QUICK START GUIDE
---
**Clinic System:** RK Dental Clinic Suite  
**Release Version:** v1.0 RC1  
**Target Group:** Receptionists, Oral Surgeons, and Dental Assistants  

---

```
================================================================================
                      RECEPTIONIST WORKFLOW (FRONT DESK)
================================================================================

  [1] REGISTER PATIENT
  └── Go to "Patient Management" -> Click "Register New Patient" -> Fill Demographics
  
  [2] CREATE APPOINTMENT & GENERATE TOKEN
  └── Go to "Appointments" -> Click "Book Appointment" -> Choose Patient, Doctor & Date
      └── System automatically generates a sequential Queue Token (e.g., Token #3)
      
  [3] ALLOCATE RECOVERY BED (Post-Op Complex Cases)
  └── Go to "Beds" -> Select any Green Vacant Bed -> Click "Assign Patient" -> Save
  
  [4] COMPILE BILL & ISSUE INVOICE
  └── Go to "Billing & Payments" -> Select Patient -> Review Linked Dental Procedures
      └── Add taxes/discounts -> Click "Create Invoice"
      
  [5] CAPTURE PAYMENT
  └── Click "Record Payment" on the invoice -> Select Method (UPI, Cash, Card) -> Print
```

---

```
================================================================================
                       DOCTOR WORKFLOW (CLINICAL ROOM)
================================================================================

  [1] FETCH PATIENT PROFILE
  └── Open "Doctor Workspace" -> Locate active patient in your assigned Token Queue
      └── Click "Start Consultation" to load clinical histories
      
  [2] DIAGNOSE ON INTERACTIVE DENTAL CHART
  └── Click "Dental Chart" tab -> View 32-Tooth visual grid
      └── Click any tooth -> Mark pathological status (e.g., Caries, Missing, RCT)
      └── System auto-syncs selected treatment plans into the billing queue
      
  [3] UPLOAD DIGITAL RADIOLOGY (RVGs / OPGs)
  └── Navigate to "Radiology & RVG" tab -> Select classification (RVG/OPG/CBCT)
      └── Choose image file -> Add clinical notes -> Click "Upload & Link"
      
  [4] ADD TREATMENT NOTES
  └── Write clinical annotations, chief complaints, and observations into the text fields
  
  [5] PRESCRIBE MEDICATIONS & COMPLETE SESSION
  └── Search common drugs -> Set dosages (e.g., BID, TID) & duration -> Click "Save"
      └── Click "Print Prescription" to deliver prescription sheet to patient
```
